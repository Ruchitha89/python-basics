# String Methods
(there are so many python string methods are there ,we can gothrough google)
- **upper():** Converts a string to uppercase
- **lower():** Converts a string to lowercase
- **strip():** Removes leading and trailing spaces from a string
- **replace(old,new):** Replaces a substring with another string.

  **Example:**
  ```python
  message="Warning!"
  print(message.upper()) #Output:WARNING!
  print(message.lower()) #Output:warning!
  print(message.strip()*2) #Output:Warning!Warning (no space between two words)its strip
  print(message.replace("Warning!","Error!"))
  ```
