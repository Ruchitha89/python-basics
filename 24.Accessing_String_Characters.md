# Accessing String Characters
You can access individual characters in a string using **indexing.** Python uses Zero-based indexing, So the first character has no index of 0.

```python
p y t h o n
0 1 2 3 4 5
```
**Example:**
```python
text="Python"
print(text[0]) #output:P
print(text[2]) #output:t
```
You can also use **negative indexing** to start counting from the end of the string.
```python
 p  y  t  h  o  n
-6 -5 -4 -3 -2 -1
```
**Example:**
```python
text="python"
print(text[-1]) #output:n
print(text[-3]) #output:h
```
