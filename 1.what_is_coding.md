What is Coding?
- Set of instructions given to the computer to perform a task.
- Imagine Person A and Person B using English, Kannada to communicate each other .But Computer can't understand this language ,it knows only binary language (0 1)
- So, developers use programming language (python/java) to communicate with computer and perform task.

How it works:	
- First developer write program ,using programming language 
- Second Interpreter converts this language into binary form
- So finally ,Computer can easily understand binary language.
