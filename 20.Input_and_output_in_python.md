# Input and Output in Python

# Input from the user:
In python, we use the **input() function** to take input from the user.The data entered by the user is always **received as a string**,so if you want to 
use it as a different data type (eg.,integer or float),you'll need to **convert it using type conversion functions** like int() or float().

**Example:**
```python
name=input("Enter your name:")
age=int(input("Enter your age:")) #Convert input to integer.
```
# Output to the Console:
The **print() function**is used to display output to the console.You can use it to display text,variables,or results of expressions.
**Example:**
```python
print("Hello,"+name+"!you are"+str(age)+"Years old.")
```
You can also use **f-strings**(formatted string literals)for more readable code:
```python
print(f"Hello,{name}!you are{age}years old.")
```
# Program:
```python
boy_name=input("Boy Name:")
boy_age=int(input("boy age:"))
girl_name=input("Girl Name:")
girl_age=int(input("girl age:"))

print(f"{boy_name}loves{girl_name}.Age Difference is{age_diff}")
```
