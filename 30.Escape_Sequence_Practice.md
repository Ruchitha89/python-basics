# Escape Sequence Practice:
Write a Python program that uses escape sequences to print the following output:
**Example:**
```python
'''
Hello
    World
This is a backslash: \
'''
```
**Solution**
```python
a="Hello\n\t world \n This is a backslash:\\"
# OR
print("Hello")
print("\t world")
print("This is a backslash:\\")
```
