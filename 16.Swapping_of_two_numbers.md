# Swap Two Variables: Write a python program that swaps the values of two variables with and without using a third variable.

**Swapping using a third variable**
```python
#Assign values
a=10
b=20
print("Before Swapping:")
print("a=",a)
print("b=",b)
#use a third variable
temp=a #temp gets 10 
a=b #a gets 20
b=temp #b gets 10
print("After swapping:")
print("a=",a)
print("b=",b)
```
**Swapping without using a third variable**
```python
a=10
b=20
print("Before swapping:")
print("a=",a)
print("b=",b)
#Swap directly
a,b=b,a
print("After Swapping:")
print("a=",a)
print("b=",b)
```

**Method 2**(without using a third variable)
```python
a=10
b=20
print("Before Swapping:")
print("a=",a)
print("b=",b)
#logic using + & -
a=a+b # a becomes 30
b=a-b # b becomes 10
a=a-b # a becomes 20
print("After Swapping:")
print("a=",a)
print("b=",b)
```
