# Membership Operator Exercise:
Write a Python program that:
- Takes a string as input from the user.
- Checks if the letter 'a' is in the string (using `in`).
- Checks if the string doesn't contain the word "Python" (using `not in`).

**Solution:**
```python
thing = input("Enter a String:")
print("Contains letter 'a':",'a' in thing) #'a' in thing -> checks if letter 'a' is present.
print("Does not contain 'python':",'python' not in thing) #'python' not in thing -> checks if word 'python' is absent.
```
