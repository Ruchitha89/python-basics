# Data types in Python
Python has various built-in data types.Some common ones are:
- **int**: For integers(eg.,-3,1,100)
- **float**: For floating-point numbers(eg.,3.14,-0.001)
- **str**: For Strings(eg.,"Hello","Python")
- **bool**: For boolean values(True or False)
- **None**: None is a special data type in Python that represents no value or empty value.
It means nothing or value is not available.
- It is not 0
- It is not False
- It is not an empty string
- It is its own data type
- **example:**
```python
  x = None
  print(type(x))
```
